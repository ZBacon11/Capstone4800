# Retro To-Do App (Windows 98 Style)

A clone of asimple Windows 98-style To-Do List web application built with HTML, CSS, and JavaScript.

## Features

- Add, edit, and remove tasks
- Mark tasks as completed individually or all at once
- Clear all tasks
- Task counter showing remaining tasks
- Notifications for task actions
- A Zack Button that displays some information about "Zack" and how many days until the semester is over
- Saving storage using `localStorage`
- Responsive design: works well on all devices and screen sizes
- Styled with [98.css](https://jdan.github.io/98.css/) for the Windows 98 look.
